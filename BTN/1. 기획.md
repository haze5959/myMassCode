# 바코드 시음노트

## 아이디어
- 바코드 인식 및 AI 라벨 스캔을 이용한 빠른 시음노트 작성
- 바코드는 유니크한 값이므로 남이 작성한 정보가 이미 있으면 빠른 작성 가능
	- 이미 작성한 노트가 있으면 추가 등록 페이지로
- 팔로우 기능도 제공
- 일단 찍고 기록은 내일하기 기능
	- 내일 아침에 알림이 오고 그 때 찍은 사진으로 기록을하게 됨
  - 로그인 과정은 기록 시점에 진행해서 카메라를 바로 띄우는게 요건임
- 내가 적어온 노트들을 하나의 링크로 바로 공유

## 디테일
- 로그인은 카카오(한국), 구글, 애플 로그인만 제공 (MVP에서는 애플 로그인만 제공)
- 바코드를 찍으면 바로 상세정보(누가 등록을 이미 했다면)로 이동
	주종, 도수, 당도, 산도 정보들이 보이며 최종 별점과 괄호에 별점 매겨진 횟수 표시
- 노트에는 별점, 향, 메모만 기입
	- 향, 메모는 선택사항
	- 심플하게 작성할 수 있도록 하는걸 추구하기 때문에 왠만해서는 추가 항목은 지양
  
## MVP 이후에 구현할 목록
- 바코드가 없는 주류 등록 기능
	- 바코드로 진입하게해야 컬렉팅 재미도 있고 그러지 않을까?
- 푸시, 알림
- 잘못된 정보 신고 기능
- 술장 기능(MVP 이후 기능)
	술병을 찍으면 자동으로 크롭해줌
	그걸 술장에 직접 드래그앤드롭
	술장 추가 및 술장 고급 디자인 유료
  - 캔? 병? 개수 및 뜯은 날짜 표시 기능
