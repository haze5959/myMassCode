## 디테일
- 로그인은 카카오(한국), 구글, 애플 로그인만 제공 (MVP에서는 애플 로그인만 제공)
- 바코드를 찍으면 바로 상세정보(누가 등록을 이미 했다면)로 이동
	주종, 도수, 당도, 산도 정보들이 보이며 최종 별점과 괄호에 별점 매겨진 횟수 표시
- 노트에는 향, 당도, 산도, 바디, 타닌 설명
	댓글도 표시
- 술장 기능
	술병을 찍으면 자동으로 크롭해줌
	그걸 술장에 직접 드래그앤드롭
	술장 추가 및 술장 고급 디자인 유료
  - 캔? 병? 개수 및 뜯은 날짜 표시 기능

### UI 디자인
- 홈화면: 네비게이션바 가운데에 QR 버튼

  2. 새로 등록된 주종
  3. 내 술장
  4. 내 최근 노트
  
- 마이메뉴: 알림, 설정, 팔로워/팔로윙 관리, 내 프로필 관리, 내 시음노트 히스토리, 좋아요 리스트, 내 술장
- 등록정보 화면: 당도, 산도, 바디, 타닌 등등은 전부 모든 데이터의 평균값으로 나옴
- 등록하기 화면:
- 술장: 위젯으로도 제공
