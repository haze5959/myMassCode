- 구독 웹훅
	- #### 단계 3: App Store Server Notifications V2 (웹훅) 연동 - ⭐️ 가장 중요
	- 유저가 앱을 지우고 설정에서 구독을 해지하거나, 다음달에 자동으로 카드가 결제되어 연장되었을 때 서버는 어떻게 알 수 있을까요? 바로 **애플 서버가 우리 서버로 매번 알림(Webhook)을 쏴줍니다.**
		1. **App Store Connect** 이동
		2. 앱 설정 -> 일반 -> 앱 정보 -> **App Store 서버 URL** 항목에 백엔드 API 주소 입력 (예: `https://api.barnote.com/webhooks/appstore`)
		3. 이후 유저의 결제와 관련된 모든 이벤트(예: `SUBSCRIBED`(최초 결제), `RENEWED`(기간 연장), `EXPIRED`(구독 만료), `REFUND`(환불))가 발생할 때마다 백엔드로 데이터가 날아옵니다.
		4. 백엔드는 날아온 데이터에서 **`appAccountToken`(1단계에서 심은 유저 UUID)** 을 읽어내어 서버 DB에서 해당 유저를 찾아 `is_premium` 상태와 만료일(Expiration Date)을 자동으로 업데이트합니다.
	1. users 에 premium_expire_at 컬럼 추가하기
		1. 메일로 알려주기
		2. GET admin/dashboard에 premium_user_count 추가
- 메인에 푸시하면 자동배포되는 파이프라인 구현하기
	- 프로덕션으로 빌드된 파일이 집컴에 배포돼서 서버 재부팅시켜야 함
