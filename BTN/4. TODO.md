		## 1. Cloudflare-DDNS 전용 앱 사용 (가장 쉬운 방법)
		Mac용으로 나온 가벼운 오픈소스 앱을 사용하면 메뉴바에서 상태를 바로 확인할 수 있습니다.
		- **추천 앱:** [Cloudflare-DDNS-Updater](https://www.google.com/search?q=https://github.com/m-p-3/Cloudflare-DDNS-Updater) (또는 유사한 앱)
		- **설정 방법:**

	    1. 3AL7zveA8Tc-uAKLDZ2j86pxO7heK61k3YsvveB7
    2. 앱을 실행하고 발급받은 **Token**과 여러분의 **도메인(`api.yourdomain.com`)**을 입력합니다.
    3. 업데이트 주기(예: 5분마다)를 설정하면 끝입니다.
        
	## 3. Mac 서버를 위한 필수 체크리스트
	Mac을 서버로 쓸 때 놓치기 쉬운 **'잠자기 방지'** 설정을 꼭 해주셔야 합니다.
	- **시스템 설정 (System Settings):**
	    - **에너지 절약(Energy Saver)** 또는 **배터리**: '디스플레이가 꺼져 있을 때 컴퓨터를 자동으로 잠자지 않게 하기' 활성화.
	    - **네트워크 깨우기(Wake for network access)**: 활성화 (외부에서 접속 시 서버가 깨어 있어야 함)
	- **방화벽(Firewall):**
	    - `시스템 설정 -> 네트워크 -> 방화벽`에서 서버 앱(Node.js 등)이 외부 연결을 허용하도록 설정되어 있는지 확인하세요.

	## 4. KT 공유기 포트포워딩 실전 (Mac 기준)
	Mac의 내부 IP 주소를 확인한 뒤(`설정 -> 네트워크`에서 확인 가능, 보통 `172.30.1.x`), KT 공유기에 접속하여 다음과 같이 연결하세
	
	|**외부 포트**|**내부 IP**|**내부 포트**|**설명**|
	|---|---|---|---|
	|**443**|**Mac의 IP**|**3000**|외부 HTTPS 요청을 Mac의 API 서버(3000번 포트)로 전달|
	### **정리하자면?**
	1. Cloudflare에서 **API 토큰**을 만든다.
	2. Mac에 **DDNS 업데이트 도구**(앱 또는 Docker)를 설치해 IP가 바뀌어도 도메인이 따라오게 만든다.
	3. KT 공유기에서 **443 포트를 내 Mac의 포트**로 열어준다.
	4. Mac이 **잠자기 모드**에 빠지지 않게 설정한다.
