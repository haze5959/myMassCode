# Front-End
- android 앱
	- ## 1. upload-keystore.jks (업로드 키스토어) 생성 방법
		이 파일은 **"내가 이 앱의 진짜 개발자다"라는 것을 증명하는 디지털 서명 파일**입니다. Android Studio를 이용해 컴퓨터에서 직접 생성합니다.
		
		### 생성 단계
		
		1. **Android Studio**를 실행하고 프로젝트를 엽니다.
		    
		2. 상단 메뉴에서 Build > Generate Signed Bundle / APK...를 선택합니다.
		    
		3. **Android App Bundle** 또는 **APK**를 선택하고 **Next**를 누릅니다.
		    
		4. **Key store path** 항목 아래에 있는 **Create new...** 버튼을 클릭합니다.
		    
		5. 새 창이 뜨면 정보를 입력합니다:
		    
		    - **Key store path:** 파일이 저장될 경로와 파일 이름 지정 (예: `/Users/유저명/Desktop/upload-keystore.jks`)
		        
		    - **Password:** 키스토어 자체의 비밀번호 설정 및 확인
		        
		    - **Alias:** 키의 별명 (기본값인 `key0`을 쓰거나 `upload-key` 등으로 지정)
		        
		    - **Password (Key):** 이 키의 비밀번호 설정 (위의 키스토어 비밀번호와 똑같이 맞춰두면 관리가 편합니다)
		        
		    - **Certificate:** `First and Last Name` 항목에 본인 이름이나 회사명을 적습니다. (나머지 칸은 비워두거나 간단히 국가코드 `82` 등을 적어도 됩니다)
		        
		6. **OK**를 누르면 지정한 경로에 `upload-keystore.jks` 파일이 생성됩니다.
