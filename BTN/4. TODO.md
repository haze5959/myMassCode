# Front-End
- 홈뷰에 심플로우들 그림자 이상함
- 상세평가 쪽 옵션 표시가 이상하고 인포로고를 그냥 타이틀이랑 붙이자
- 바코드 통해서 가면 아래에 노트 리스트가 안나옴
- 홈에서 노트 뷰올로 가면 달력표시 안가게 수정 필요
- 홈에서 맨 위 버튼 누르면 아무동작 안함
- 홈에서 제품 뷰올 들어가면 타이틀이라도 최신 제품이라 나오게 하기
- 노트에 주류 이모지 표시 그냥 이미지 상단에 넣는게 나을거 같음
- 미니 노트 로우에 왜 제품 이미지가 안보이지?
- 마이페이지 처음 진입시 유저정도 연속 2번 호출함
	- 로그아웃 버튼 좀 연하게해라
- exceedMaxCount: ai 사용 최대한도 도달 / failedToAnalyzeImage: ai 제품 인식 실패 처리하기
- 닉네임 1자인데도 저장버튼 활성화 됨
	- 닉네임 변경 안먹힘, intro는 변경 먹힘
- 달력 다크모드 대응하기
	- 달력 첫 집입할때는 시음기록 안보임
- 좋아요 제품 없을때 진입하면 무한로딩
- 제품 검색에 찾으시는 제품이 없다면 새로 등록하라는 버튼 보이게하기
- 한 노트, 찜한 제품 페이지 타이틀 바꿔라
	- 검색 필터링 그대로 사용하기
- 제품 직접등록일 경우,  바노트팀에서 검토하고 수정하니 부담없이 작성하라고 알려주기
- 이미지 편집모드 이상함
- 바코드 스캔버튼 아이콘 ai스러운 아이콘으로 변경하는게 좋을듯
- 스캔 후 무조건 홈으로 가는것보다 이전 탭으로 가는게 좋을것 같음
- 프로필 이미지 기능
- 작성자를 클릭하면 작성자의 노트 리스트가 나오는 기능
- 홈이 아니라 피드? 가 맞는거 같음
	- 마지막 접속 탭을 앱을 다시 켯을때 바로 이동시키기
- 시음노트 예약 리스트를 누르면 지금 바로 작성할거냐고 알림이 띄워지게 하기
- 검색 바로바로 입력 기능 제공하기
- 어드민 페이지(WEB)
	- 기술
		- **Frontend:** TypeScript + **React**
		- **Admin Library:** **Refine** (또는 React-admin)
		- **UI Component Library:** **MUI** 또는 **Ant Design** (어드민에 필요한 모든 UI 컴포넌트 제공)
	- auth0로 구글 로그인만 붙이기
		- 토큰 쿠키가 없이 웹페이지 요청하면 로그인으로 리다이렉트
	- admin_sub 값 env에 하드코딩된걸로 체크
	- 신규 추가 제품 보기
	- 신규 노트 보기
	- 제품 정보 수정하기
		- 제품 이름 변경하면 embedding 값도 갱신해야됨
	- 신고 리스트 보기
	- 제품 merge 기능
		- product_id로 매핑된 product_images, notes, favorites, barcodes... 를 to_product_id로 변경 후 제거
	- 제품 메인 이미지(product_images에 product_id만 매핑되어 있는거) 변경
		- id는 그대로 두고 이미지만 바꿔치기
	- 관리 방향성
		- 중복되고 명칭이 확실한 제품은 머지
		- 사용자가 직접 등록한 제품인데 이름이 영어가 아닌경우에는 그냥 놔둔다
			- 직접 등록 제품은 왼만하면 건들지 않기
			- 같은 언어권끼리는 확실한 명칭인 제품인 경우 머지
		- 노트가 작성된 제품 우선으로 정리
