- ### GitHub에 시크릿(Secrets) 등록하기

	GitHub 코드가 퍼블릭으로 공개되어 있더라도, API 키 같은 민감한 정보는 안전하게 숨겨야 합니다.
	
	1. 배포할 GitHub 레포지토리(저장소)로 이동합니다.
	    
	2. 상단 탭에서 **[Settings]** ➔ 왼쪽 메뉴에서 **[Secrets and variables]** ➔ **[Actions]** ➔ **[New repository secret]**을 클릭합니다.
	    
	3. 1단계에서 얻은 정보를 바탕으로 다음 4개의 시크릿을 추가해 줍니다.
	    
	    - `R2_ACCESS_KEY_ID` : (발급받은 Access Key ID)
	        
	    - `R2_SECRET_ACCESS_KEY` : (발급받은 Secret Access Key)
	        
	    - `R2_ACCOUNT_ID` : (Cloudflare Account ID 영문/숫자 조합)
	        
	    - `R2_BUCKET_NAME` : (생성한 버킷 이름, 예: `my-react-app-bucket`)
