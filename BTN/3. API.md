# 공통 규칙
## common header
```json
{"authorization": "Bearer ACCESS_TOKEN"}
```
## common response model
```json
// success
{
	"result": true,
	"data": data model
}

// fail
{
	"result": false,
	"error": {
		"code": 9999,
		"msg": "unknown"
	}
}
```

# notes
## post
### path: api/notes/
#### body
```json
{"product_id": "product_id",
"body": "body",
"rating": 10,
"selected_flavors": [1, 2, 3],
"image_ids": [ProductImage id],
"is_public": true}
```
#### response data
- Note model

## get
### path: notes/:id
#### response data
```json
{"note": Note model,
"product": Product model,
"user": User model(Optional),
"image_ids": [ProductImage id],
"product_image_id": [uuid](Optional)(image_ids가 없을 경우에만 존재),
"flavors": [Int](Optional)}
```

### path: notes/
#### query
- ids: "{uuid},{uuid},{uuid}" (이 값이 있으면 페이징 안하고 바로 응답)
- page
- per
- product_id(optional)
#### response data
```json
[
  {"note": Note model,
  "product": Product model,
  "user": User model(Optional),
  "image_ids": [ProductImage id](최대 3개),
  "product_image_id": [uuid](Optional)(image_ids가 없을 경우에만 존재)}
]
```

### path: notes/user/:id
#### query
- page
- per
- order_by: "registered"(default), "rating"
#### response data
```json
[
  {"note": Note model,
  "product": Product model,
  "user": User model(Optional),
  "image_ids": [ProductImage id](최대 3개),
  "product_image_id": [uuid](Optional)(image_ids가 없을 경우에만 존재)}
]
```

### path: api/notes/rating
#### query
- page
- per
- rating: Int (0~10)
#### response data

[
  {"note": Note model,
